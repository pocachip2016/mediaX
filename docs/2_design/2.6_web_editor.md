# 2.6 디자인 에디터 (웹 기반)

**목적:** AI 생성 시안을 디자이너가 브라우저에서 직접 미세 조정하고, 멀티 디바이스 프리뷰로 확인 후 DAM에 저장.

---

## 접근 시나리오

| 진입 경로 | 대상 |
|----------|------|
| DAM 에셋 상세 → "에디터에서 열기" | 기존 에셋 수정 |
| 배치 대시보드 → 개별 시안 선택 → "수정" | 배치 결과 미세 조정 |
| 신규 에셋 생성 → 템플릿 선택 → 편집 | 수동 제작 시작 |
| 템플릿 관리 → "템플릿 편집" | 팀장·시니어 디자이너 전용 |

---

## 편집 기능

### 텍스트 편집

| 기능 | 상세 |
|------|------|
| 제목 텍스트 수정 | 자유 입력, 줄바꿈 지원 |
| 폰트 크기 조정 | 허용 범위 내 슬라이더 |
| 텍스트 색상 | KT CI 허용 팔레트 내 선택 |
| 텍스트 위치 | 드래그 이동 (safe-zone 내 제한) |
| 카피 문구 | 이벤트·프로모션용 자유 입력 영역 |

> ⚠️ 폰트 패밀리 변경 불가 — KT CI 고정 폰트 적용

### 이미지 편집

| 기능 | 상세 |
|------|------|
| 이미지 교체 | DAM에서 검색 후 선택 또는 직접 업로드 |
| 크롭·위치 조정 | 드래그 크롭, 확대·축소 |
| 밝기·대비 조정 | 슬라이더 (±30% 범위) |
| 배경 색상·그라디언트 | 허용 팔레트 내 선택 |

### 레이아웃 편집

| 기능 | 상세 |
|------|------|
| 레이어 순서 | 드래그앤드롭 레이어 패널 |
| 레이어 숨김/표시 | 눈 아이콘 토글 |
| 요소 정렬 | 가운데·좌우·상하 정렬 가이드라인 |

> ⚠️ 로고·등급 배지 레이어: 이동 불가 (CI 고정)

## 멀티 디바이스 프리뷰

- 편집 중 실시간 프리뷰: TV / 모바일 / 웹 3개 탭 동시 확인
- 각 슬롯 코드별 렌더링 결과 미리보기 (실제 픽셀 크기 기준)
- 다크모드 배경 시뮬레이션 (지니TV UI 참고)

---

## 저장 및 발행 정책

```
편집 완료
  → "임시 저장"   → DAM status 유지 (draft 상태), 수정 이력 기록
  → "검수 요청"   → 브랜드 검수(2.5) 자동 실행
      → 통과  → DAM status: review, 팀장 승인 대기
      → 실패  → 실패 항목 팝업 표시, 재편집 유도
  → "승인" (팀장) → DAM status: approved → published
```

- **자동 저장:** 30초마다 임시 저장 (브라우저 닫아도 복구 가능)
- **버전 분기:** "다른 버전으로 저장" 시 기존 버전 유지 + 신규 버전 생성

---

## 권한 체계

| 역할 | 편집 가능 항목 |
|------|---------------|
| 디자이너 (정직원) | 텍스트·이미지·레이아웃 전체 |
| 프리랜서 | 텍스트·이미지만 (레이아웃·레이어 잠금) |
| 팀장 | 전체 + 템플릿 편집 모드 |
| 편성팀 | 읽기 전용 프리뷰만 |

---

## 기술 스펙

- **프레임워크:** Next.js 14 (기존 프론트엔드 통합)
- **캔버스 렌더링:** Fabric.js 또는 Konva.js (레이어 기반 편집)
- **이미지 처리:** Sharp (서버사이드 리사이즈·크롭)
- **저장 방식:** 편집 상태 JSON → 서버 저장 → 최종 렌더링 시 이미지 export

## Phase별 구현 범위

| Phase | 내용 |
|-------|------|
| Phase 1 (MVP) | 텍스트·이미지 기본 편집, 단일 슬롯 프리뷰, 임시저장 |
| Phase 2 | 멀티 슬롯 동시 프리뷰, 레이어 패널, 브랜드 검수 연동 |
| Phase 3 | 실시간 협업 편집 (디자이너 ↔ 팀장), 코멘트 기능 |
