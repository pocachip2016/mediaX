# 2.3 스마트 템플릿 관리

**목적:** 장르·콘텐츠 유형·시즌별 최적 템플릿을 자동 추천하고, 성과 데이터 기반으로 랭킹을 지속 개선.

---

## 템플릿 구조

### 분류 체계

```
템플릿
├── 슬롯 유형 (tv_poster_lg / mobile_banner_main / …)
├── 장르 (드라마 / 영화 / 예능 / 다큐 / 애니 / 스포츠)
├── 콘텐츠 유형 (단편 / 시리즈 / 이벤트)
├── 분위기 태그 (긴장감 / 로맨스 / 코미디 / 액션 / 힐링)
└── 시즌 태그 (봄·여름·가을·겨울 / 명절 / 창립기념)
```

### 템플릿 구성 요소

| 요소 | 설명 | 편집 가능 여부 |
|------|------|----------------|
| 배경 레이어 | 그라디언트 또는 이미지 영역 | 에디터에서 교체 가능 |
| 이미지 safe-zone | 포스터·스틸컷 배치 영역 | 크기·위치 조정 가능 |
| 제목 텍스트 | 폰트·크기·색상 프리셋 | 폰트 변경 불가 (CI 고정) |
| 장르/등급 배지 | 등급 표시 아이콘 | 자동 삽입 |
| 방송사 로고 | KT/지니TV CI | 위치 고정, 크기만 조정 |
| 자막/카피 영역 | 이벤트성 문구 (선택) | 텍스트 자유 입력 |

---

## AI 자동 추천 로직

```
[입력]
  content_id → 메타데이터 AX에서 장르, 감성태그, 인물 수, 콘텐츠 유형 조회

[추천 점수 계산]
  score = 0.4 × 장르매칭 + 0.3 × 성과점수 + 0.2 × 분위기매칭 + 0.1 × 최신성

[출력]
  상위 3개 템플릿 반환 (디자이너 선택 또는 배치 시 자동 1위 적용)
```

### 성과 기반 랭킹 업데이트

- **성과 지표:** 통계 AX에서 CTR(클릭률), 재생 전환율 수신
- **업데이트 주기:** 매주 월요일 새벽 3시 자동 배치
- **반영 방식:** `performance_score` 가중 평균 (신규 지표 30% + 기존 70%)
- **최소 노출 보정:** 노출 50회 미만 템플릿은 랭킹 제외 (데이터 불충분)

---

## 템플릿 등록 및 관리

### 신규 템플릿 등록 절차

```
디자이너 작성 (웹 에디터) → 팀장 승인 → 장르·슬롯 태깅 → 활성화
```

- 드래그앤드롭 에디터(2.6)에서 레이어 구성 후 저장
- 저장 시 자동으로 슬롯별 미리보기 썸네일 생성

### 버전 관리

- 템플릿 수정 시 버전 자동 증가 (v1 → v2)
- 이전 버전으로 생성된 에셋은 해당 버전 참조 유지
- 템플릿 비활성화 시 신규 배치에서만 제외 (기존 에셋 영향 없음)

### 운영 정책

| 항목 | 기준 |
|------|------|
| 슬롯당 최소 활성 템플릿 | 3개 이상 유지 |
| 성과 하위 10% 템플릿 | 분기별 비활성화 검토 |
| 이벤트 전용 템플릿 | 이벤트 종료 후 자동 만료 |
| 신규 템플릿 테스트 기간 | 2주 (성과 수집 후 랭킹 반영) |
