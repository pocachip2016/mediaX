## 에셋 파일 규격

| 항목 | 기준 |
|------|------|
| 포맷 | JPG (사진계), PNG (투명 레이어), WebP (웹 최적화) |
| 원본 저장 | PNG 또는 PSD (비손실, 마스터 파일) |
| 색상 모드 | RGB / sRGB |
| 최대 파일 크기 | JPG: 500KB / PNG: 2MB / WebP: 300KB |
| 해상도 (DPI) | 72 DPI (화면 출력 전용) |
| 네이밍 규칙 | `{content_id}_{slot_type}_{device}_{version}.{ext}` |

---

## 슬롯 코드 체계

```
slot_code = {device}_{slot_type}_{size}

예시:
  tv_banner_main      → TV 메인 배너 (1920×540)
  tv_poster_lg        → TV 포스터 대형 (270×360)
  tv_poster_sm        → TV 포스터 소형 (180×240)
  mobile_banner_main  → 모바일 메인 배너 (750×400)
  web_banner_main     → 웹 메인 배너 (1280×460)
```

## 렌더링 우선순위

배치 생성 시, 하나의 콘텐츠에 대해 아래 순서로 자동 렌더링:

1. TV 포스터 (대) — 가장 많이 노출
2. TV 메인 배너
3. 모바일 포스터 (대)
4. 웹 포스터 (대)
5. 나머지 슬롯 (요청 또는 Phase 2 자동화)
