# 2.5 브랜드 가이드라인 자동 검수

**목적:** KT/지니TV CI 기준에 따라 생성된 에셋을 자동으로 검수하고, 미통과 항목은 자동 보정 또는 담당자 알림으로 처리.

---

## 검수 항목 및 기준

### 1. 로고 (KT/지니TV CI)

| 검사 항목 | 기준 | 판정 방식 |
|----------|------|----------|
| 로고 존재 여부 | 슬롯별 필수 위치에 로고 삽입 확인 | 템플릿 레이어 존재 체크 |
| 로고 크기 | 슬롯 단변의 8%~15% 범위 | 픽셀 측정 |
| 로고 위치 | 슬롯별 safe-zone 내 지정 영역 | 좌표 범위 체크 |
| 로고 가독성 | 배경 대비 명도 차 30 이상 | RGB 평균 비교 |
| 로고 왜곡 | 비율 변형 없음 | 원본 비율 대비 오차 ±2% 이내 |

### 2. 폰트

| 검사 항목 | 기준 |
|----------|------|
| 허용 폰트 | KT 지정 폰트 목록 내 (예: KT헬로체, NotoSansKR 등) |
| 최소 폰트 크기 | TV: 24px 이상 / 모바일: 16px 이상 |
| 제목 가독성 | 배경 대비 명도 차 40 이상 |
| 폰트 혼용 | 슬롯 내 최대 2가지 폰트 패밀리 |

### 3. 컬러

| 검사 항목 | 기준 |
|----------|------|
| KT 브랜드 컬러 사용 | 주조색이 KT Red (#E60012) ±10% 범위 초과 사용 금지 |
| 배경 컬러 | 순수 흰색(#FFFFFF) 배경 단독 사용 금지 (그라디언트 권장) |
| 텍스트-배경 대비 | WCAG AA 기준 (4.5:1 이상) |
| 채도 과다 | 단일 색조 채도 100% 전면 사용 금지 |

### 4. 등급 표시

| 검사 항목 | 기준 |
|----------|------|
| 등급 배지 존재 | 모든 VOD 에셋에 시청 등급 아이콘 필수 |
| 등급 위치 | 좌상단 또는 우상단 고정 영역 |
| 등급 크기 | 슬롯 단변의 5% 이상 |
| 등급 정확성 | 메타데이터 AX의 `content_rating` 값과 일치 여부 |

---

## 자동 보정 (Auto-fix)

미통과 항목 중 아래 케이스는 자동 보정 1회 시도:

| 미통과 항목 | 자동 보정 방법 |
|------------|--------------|
| 로고 크기 이탈 | 지정 범위 내 자동 리사이즈 |
| 로고 가독성 부족 | 로고 하단에 흰색 반투명 패드 자동 삽입 |
| 텍스트-배경 대비 부족 | 텍스트 영역에 그림자 또는 반투명 오버레이 자동 추가 |
| 등급 배지 누락 | 메타데이터 기반 자동 삽입 |
| 등급 배지 크기 이탈 | 자동 리사이즈 |

자동 보정 후 재검수 → 통과 시 DAM 등록, 재실패 시 `brand_check_failed` 상태로 담당자 알림.

---

## 검수 결과 처리 흐름

```
생성 완료 에셋
  → 브랜드 검수 실행 (brand_check_results 저장)
  → overall_pass = true  → DAM draft 등록
  → overall_pass = false → 자동 보정 시도
      → 보정 후 재검수 통과 → DAM draft 등록 (보정 이력 기록)
      → 재검수 실패       → batch_job_item.status = failed
                            → 담당자 알림 (항목별 실패 사유 포함)
```

---

## 수동 검수 보완

자동 검수가 감지하기 어려운 항목은 담당자 수동 확인 대상:

- 이미지 내 부적절한 콘텐츠 (폭력·선정성)
- 경쟁사 로고·브랜드 노출
- 제목 오탈자
- 아티스트·배우 이름 표기 오류

## Phase별 구현 범위

| Phase | 내용 |
|-------|------|
| Phase 1 (MVP) | 4개 항목 기본 검수 (로고 존재·크기·등급 배지), 수동 확인 중심 |
| Phase 2 | 전체 항목 자동 검수, 자동 보정 로직 구현 |
| Phase 3 | ML 기반 시각적 품질 점수 추가 (선명도·구도·밸런스) |
