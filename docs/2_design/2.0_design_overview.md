# 2.0 디자인 AX 개요

**미션:** AI 이미지 자동 생성 + 템플릿 대량 생산 + IPTV VOD 에셋 통합 관리
(월 500~2,000건, KT/지니TV CI 준수). TV/모바일/웹 멀티 디바이스 대응.

---

## 운영 맥락

- **에셋 성격:** 영화·시리즈 포스터, VOD 프로모션 배너, GUI 슬롯별 variation, 이벤트 이미지(추석·창립기념일 등) 혼재
- **요청 주체:** CP사 또는 KT 내부 — 현재 이메일 요청, 추후 CP 포털 연동
- **제작 주체:** 디자인팀 내부 (정직원 + 프리랜서)
- **승인:** 디자인팀 내부 (팀장 최종)
- **게시:** 편성팀 별도 운영 — DAM이 디자인팀↔편성팀 간 게이트 역할

---

## 모듈 구성 (9개 섹션)

| 섹션 | 설명 |
|------|------|
| 2.1 사이즈 체계 | GUI 슬롯별 × 디바이스별 규격 정의 |
| 2.2 AI 이미지 자동 생성 | 템플릿 기반 자동 생성 (주력) + 생성형 AI (보조) |
| 2.3 스마트 템플릿 관리 | 장르·유형·시즌별 템플릿, 성과 기반 랭킹 |
| 2.4 대량 생산 배치 | Celery 비동기, 일괄 승인·재생성 |
| 2.5 브랜드 가이드라인 검수 | 로고·폰트·컬러·등급 자동 체크 |
| 2.6 디자인 에디터 (웹) | AI 시안 미세 조정, 멀티 디바이스 프리뷰 |
| **2.7 DAM (에셋 관리)** | **에셋 중심 통합 관리, VOD 연계, AI 확장 대비** |
| 2.8 연동 포인트 | 타 모듈 API 연동 |
| 2.9 DB 스키마 | 12개 테이블 |

## Phase 계획

| Phase | 주요 내용 | 완료 기준 |
|-------|-----------|-----------|
| Phase 1 (MVP) | 이메일 요청 → DAM 수동 등록, 워크플로우·RBAC, 기존 파일 마이그레이션 | DAM 수동 업로드·승인·편성팀 전달 동작 / 기존 에셋 100% 마이그레이션 |
| Phase 2 | 인제스트 AX 스틸컷 자동 등록, AI 이미지 생성 DAM 연동, 배치 시스템 가동 | 브랜드 검수 자동 통과율 80% 이상 / 배치 100건 30분 이내 처리 |
| Phase 3 | 웹 크롤링 포스터 수집, CP 포털 직접 업로드, 성과 기반 템플릿 랭킹 자동화 | CTR 기반 템플릿 추천 적용 / CP 포털 직접 요청 처리 가능 |
