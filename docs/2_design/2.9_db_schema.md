# 2.9 DB 스키마 (12개 테이블)

---

## 테이블 목록

| # | 테이블명 | 설명 |
|---|----------|------|
| 1 | `design_assets` | 에셋 마스터 |
| 2 | `asset_versions` | 버전 이력 |
| 3 | `asset_slots` | 슬롯 규격 정의 |
| 4 | `design_templates` | 템플릿 마스터 |
| 5 | `template_slots` | 템플릿-슬롯 매핑 |
| 6 | `batch_jobs` | 배치 생성 작업 |
| 7 | `batch_job_items` | 배치 개별 항목 |
| 8 | `brand_check_results` | 브랜드 검수 결과 |
| 9 | `approval_history` | 승인/반려 이력 |
| 10 | `dam_publish_log` | DAM 게시 로그 |
| 11 | `asset_performance` | 에셋 성과 데이터 |
| 12 | `design_requests` | 디자인 요청 원장 |

---

## 주요 테이블 스키마

### design_assets (에셋 마스터)
```
id              UUID        PK
content_id      VARCHAR     FK → catalog.contents.id
slot_code       VARCHAR     슬롯 코드 (예: tv_poster_lg)
device          VARCHAR     tv | mobile | web
file_url        TEXT        원본 파일 URL
cdn_url         TEXT        CDN 배포 URL
status          VARCHAR     draft | review | approved | rejected | published | archived
source          VARCHAR     manual | ai_template | ai_generative | ingest_auto
version         INT         현재 버전 번호
created_by      VARCHAR     생성자 (user_id)
approved_by     VARCHAR     승인자 (user_id, nullable)
created_at      TIMESTAMP
updated_at      TIMESTAMP
```

### asset_versions (버전 이력)
```
id              UUID        PK
asset_id        UUID        FK → design_assets.id
version         INT         버전 번호
file_url        TEXT        해당 버전 파일 URL
created_by      VARCHAR
created_at      TIMESTAMP
```

### asset_slots (슬롯 규격 정의)
```
slot_code       VARCHAR     PK (예: tv_banner_main)
device          VARCHAR     tv | mobile | web
slot_type       VARCHAR     banner | poster | thumbnail | event | category_title
width           INT         픽셀 너비
height          INT         픽셀 높이
format          VARCHAR     jpg | png | webp
max_size_kb     INT         최대 파일 크기 (KB)
```

### design_templates (템플릿 마스터)
```
id              UUID        PK
name            VARCHAR     템플릿명
genre           VARCHAR     장르 태그
content_type    VARCHAR     movie | series | event
slot_code       VARCHAR     FK → asset_slots.slot_code
thumbnail_url   TEXT        미리보기 이미지
performance_score FLOAT     성과 기반 랭킹 점수
is_active       BOOLEAN
created_at      TIMESTAMP
```

### batch_jobs (배치 생성 작업)
```
id              UUID        PK
status          VARCHAR     queued | running | done | failed
total_count     INT         전체 항목 수
done_count      INT         완료된 항목 수
failed_count    INT         실패 항목 수
created_by      VARCHAR
created_at      TIMESTAMP
finished_at     TIMESTAMP   nullable
```

### batch_job_items (배치 개별 항목)
```
id              UUID        PK
batch_id        UUID        FK → batch_jobs.id
content_id      VARCHAR
slot_code       VARCHAR
template_id     UUID        nullable (AI 자동 선택 시 null → 후 기록)
asset_id        UUID        nullable (생성 후 기록)
status          VARCHAR     queued | done | failed
error_msg       TEXT        nullable
```

### brand_check_results (브랜드 검수 결과)
```
id              UUID        PK
asset_id        UUID        FK → design_assets.id
logo_ok         BOOLEAN
font_ok         BOOLEAN
color_ok        BOOLEAN
rating_ok       BOOLEAN
overall_pass    BOOLEAN
detail          JSONB       항목별 상세 결과
checked_at      TIMESTAMP
```

### approval_history (승인/반려 이력)
```
id              UUID        PK
asset_id        UUID        FK → design_assets.id
action          VARCHAR     submitted | approved | rejected
actor           VARCHAR     처리자 user_id
comment         TEXT        반려 사유 (nullable)
created_at      TIMESTAMP
```

### dam_publish_log (DAM 게시 로그)
```
id              UUID        PK
asset_id        UUID        FK → design_assets.id
published_at    TIMESTAMP
published_by    VARCHAR
cdn_url         TEXT        배포된 CDN URL
destination     VARCHAR     programming | distribution | cp_portal
```

### design_requests (디자인 요청 원장)
```
id              UUID        PK
content_id      VARCHAR
slot_codes      JSONB       요청된 슬롯 목록
requester       VARCHAR     요청자 (cp_id 또는 내부 user_id)
request_source  VARCHAR     email | cp_portal | internal
priority        VARCHAR     normal | urgent
status          VARCHAR     received | in_progress | done
due_date        DATE        nullable
created_at      TIMESTAMP
```

### template_slots (템플릿-슬롯 매핑)
```
id              UUID        PK
template_id     UUID        FK → design_templates.id
slot_code       VARCHAR     FK → asset_slots.slot_code
layer_config    JSONB       레이어 구성 (safe-zone 좌표, 텍스트 영역, 로고 위치 등)
render_order    INT         멀티 슬롯 렌더링 우선순위
is_primary      BOOLEAN     대표 슬롯 여부 (썸네일 미리보기 기준)
```

### asset_performance (에셋 성과 데이터)
```
id              UUID        PK
asset_id        UUID        FK → design_assets.id
slot_code       VARCHAR     노출된 슬롯 코드
period_start    DATE        집계 시작일 (주 단위)
period_end      DATE        집계 종료일
impressions     BIGINT      노출 수
clicks          BIGINT      클릭 수
ctr             FLOAT       클릭률 (clicks / impressions)
play_starts     BIGINT      재생 시작 수
play_rate       FLOAT       재생 전환율 (play_starts / clicks)
template_id     UUID        FK → design_templates.id (성과 랭킹 역산용)
collected_at    TIMESTAMP   통계 AX 수신 시각
```
