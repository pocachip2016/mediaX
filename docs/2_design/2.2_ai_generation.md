# 2.2 AI 이미지 자동 생성

**두 가지 방식:** 템플릿 기반 자동(주력) + AI 생성형(보조)

---

## A. 템플릿 기반 자동 생성 (주력)

### 파이프라인

```
[메타데이터 수신]
  → 콘텐츠 유형·장르·감성태그 파싱
  → 템플릿 자동 선택 (2.3 성과 랭킹 기반)
  → 슬롯별 레이아웃 로드
  → 이미지 소스 확보 (스틸컷 / 포스터 원본 / 웹 크롤링)
  → 이미지 자동 배치 (얼굴 중심 크롭, 피사체 위치 감지)
  → 타이포그래피 적용 (제목·장르·등급·방송사 CI)
  → KT/지니TV CI 자동 적용
  → 멀티 슬롯 렌더링 (tv_poster_lg, mobile_banner_main 등)
  → 시안 3~5개 생성
  → 브랜드 검수 (2.5)
  → DAM draft 등록
```

### 이미지 소스 우선순위

| 순위 | 소스 | 비고 |
|------|------|------|
| 1 | 인제스트 AX 스틸컷 (원본) | Phase 2 자동 수신 |
| 2 | CP사 제공 포스터 원본 | DAM 수동 업로드 |
| 3 | 웹 크롤링 공식 포스터 | Phase 3 |
| 4 | AI 생성형 이미지 | 소스 없을 때 fallback |

### 자동 배치 로직

- **얼굴 감지:** AWS Rekognition or Google Vision → 주인공 얼굴 위치 파악 → 템플릿 safe-zone 내 자동 배치
- **여백 보정:** 슬롯 비율 맞춤 크롭 시 피사체 잘림 방지 (center-aware crop)
- **배경 확장:** 원본보다 슬롯이 넓을 경우 AI 인페인팅으로 배경 자동 확장

### 시안 선택 기준

- 3~5개 시안 자동 생성 후 브랜드 검수 통과한 것만 DAM 등록
- 검수 통과 시안이 1개 이하이면 배치 담당자에게 알림 → 수동 확인 요청
- 편성팀 또는 디자이너가 최종 시안 1개 선택

---

## B. AI 생성형 보조

### 사용 시나리오

- 이미지 소스가 전혀 없는 신규 콘텐츠
- 이벤트성 특수 이미지 (명절, 창립기념일 등)
- 실험적 컨셉 시안 필요 시 디자이너 요청

### 지원 모델

| 모델 | 특성 | 주 용도 |
|------|------|---------|
| DALL-E 3 (OpenAI) | 텍스트 지시 충실, 안정적 품질 | 배너·이벤트 이미지 |
| Stable Diffusion (API) | 커스터마이징 자유도 높음, 빠른 속도 | 대량 변형 시안 |
| Midjourney (수동) | 고퀄리티 아트워크 | 프리미엄 콘텐츠 (수동) |

### 프롬프트 전략

```
[기본 프롬프트 구조]
"{제목}, {장르} 분위기, {감성태그}, IPTV VOD 포스터 스타일,
 {슬롯 비율} 구도, 한국 OTT 디자인, 고해상도, 상업적 품질"

[금지어 필터]
- 저작권 인물명, 실존 브랜드 로고, 폭력·선정적 키워드 자동 제거

[품질 파라미터]
- 해상도: 최소 1024px 단변 이상
- style_preset: digital-art 또는 photographic
- cfg_scale: 7~10 (충실도·창의성 균형)
```

### 생성 후 처리

- AI 생성 이미지도 브랜드 검수(2.5) 필수 통과
- `source = ai_generative` 태그로 DAM 등록
- 저작권 관련 고지: 생성 AI 출처 메타 기록
