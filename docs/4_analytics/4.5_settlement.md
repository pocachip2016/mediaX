# 4.5 CP사 정산 시스템

> **비즈니스 모듈:** M6. 정산 / 통계 시스템

## 정산 파이프라인

자동집계 → CP별 비율적용 → 홀드백 차등 → 정산서 자동생성 → 검토/승인 → 발송 → 이의신청

---

## 정산 유형별 처리

| 정산 유형 | 매출 기준 | 정산 주기 |
|----------|---------|---------|
| PPV (건당구매) | 콘텐츠별 건당 판매 매출 | 월별 |
| PPM (월정액) | 구독 매출 × 시청 비율 배분 | 월별 |
| 채널 라이선스 | 계약 고정 금액 | 계약 주기 |
| 수익 배분형 RS | 매출 × 배분 비율 | 월별 |

---

## ERP 연동 (신규 — M6 비즈니스 요구사항)

### 개요

정산 완료 데이터를 ERP(Enterprise Resource Planning) 시스템과 자동 연동하여 회계 처리, 세금계산서 발행, 비용 계정 배분을 자동화한다.

### ERP 연동 항목

| 항목 | ERP 매핑 | 설명 |
|------|---------|------|
| PPV 매출 정산 | 수익 계정 (매출채권) | 콘텐츠별 매출 계정 코드 |
| PPM 매출 정산 | 수익 계정 (구독 매출) | 월정액 구독 매출 |
| 채널 매출 정산 | 수익 계정 (채널 라이선스) | 채널 계약 매출 |
| CP 정산 지급 | 비용 계정 (콘텐츠 원가) | CP 지급 비용 |
| 코스트센터 (CC) | 사업부별 CC 코드 | 편성/운영팀별 비용 배분 |

### 세금계산서 자동 발행

```
정산 확정 (월말 D+5)
    │
    ▼
ERP 연동 API 호출 → 정산 데이터 전송
    │
    ├── 세금계산서 자동 발행 요청
    │       ├── 공급자: (자사)
    │       ├── 공급받는자: CP사 사업자등록 정보
    │       ├── 공급가액: 정산 금액
    │       └── 부가세: 10%
    │
    ├── ERP 분개 자동 생성
    │       ├── 차변: 콘텐츠 원가 (비용 계정)
    │       └── 대변: 미지급금 (부채 계정)
    │
    └── 국세청 전자세금계산서 발행
    │
    ▼
CP사에 세금계산서 이메일 발송
    │
    ▼
지급 일정 등록 → 지급 완료 후 ERP 결제 처리
```

### ERP 연동 스펙

- **연동 방식:** REST API (ERP 시스템별 어댑터 구현)
- **지원 ERP:** SAP, Oracle ERP, 더존 iCUBE (어댑터 플러그인)
- **전송 주기:** 정산 확정 즉시 (실시간) + 일일 배치 정합성 검증
- **오류 처리:** 전송 실패 시 재시도 3회 → 담당자 알림

### 스크린별/CP별/상품별 매출 통계 (M6 비즈니스 요구사항)

| 분류 기준 | 제공 지표 |
|----------|---------|
| 스크린별 | GTV / 모바일 / 웹 별 매출 분포 |
| CP별 | CP 기여 매출, 정산 비율, 지급액 |
| 상품별 | PPV / PPM / 채널별 매출 비중 |
| 기간별 | 일/주/월/분기/연도별 추이 |
| 장르별 | 장르 카테고리별 매출 성과 |
