# 10.1 플랫폼별 배포 현황 관리

## 개요

인코딩 및 CDN 업로드 완료 이후, 각 서비스 플랫폼(GTV, 모바일 앱, 웹 등)에서 콘텐츠가 정상 노출되는지 통합 점검하는 기능이다.

---

## 1. GTV / 다중 단말 배포 점검

### 점검 대상 단말

| 단말 유형 | 플랫폼 | 점검 항목 |
|----------|--------|----------|
| 지니TV (GTV) | IPTV STB | 영상 재생, 메타 표시, 등급 이미지 |
| 모바일 앱 | iOS / Android | 재생 가능 여부, 화질 선택, DRM 인증 |
| 웹 브라우저 | PC / 태블릿 | 스트리밍 재생, 자막, 썸네일 로딩 |
| OTT 연동 | 외부 OTT | API 응답 정합성, 메타 동기화 |

### 배포 점검 워크플로우

```
CDN 업로드 완료 신호 수신
    │
    ▼
단말별 테스트 재생 자동 실행 (Smoke Test)
    ├── 재생 성공 → 배포 상태 "정상" 표시
    └── 재생 실패 → 알림 발송 + 담당자 수동 확인 요청
    │
    ▼
메타데이터 정합성 검증
    ├── 상품명, 등급, 가격, 썸네일 URL 일치 여부 확인
    └── 불일치 시 → 오류 로그 기록 + 재배포 요청
    │
    ▼
배포 현황 대시보드 업데이트
```

### 배포 현황 대시보드

- 콘텐츠별 단말 배포 상태 매트릭스 (정상/오류/미배포)
- 최근 24시간 배포 성공률
- 오류 콘텐츠 목록 및 원인 분류
- 단말별 평균 배포 지연 시간

## 2. 영등위 이미지 외부 배포 상태

### 개요

영상물등급위원회(영등위)에서 발급받은 등급 고지 이미지(15세 이용가, 청소년 관람불가 등)가 각 플랫폼에 정상 표시되는지 확인한다. 미표시 또는 잘못된 이미지 사용 시 법적 문제가 발생할 수 있다.

### 영등위 이미지 유형

| 등급 | 이미지 코드 | 표시 의무 |
|------|-----------|---------|
| 전체 이용가 | G | 의무 |
| 12세 이용가 | 12 | 의무 |
| 15세 이용가 | 15 | 의무 |
| 청소년 관람불가 | 18 | 의무 |
| 심의 면제 | EXEMPT | — |

### 점검 항목

- 각 플랫폼 노출 화면에서 등급 이미지 렌더링 확인
- 이미지 URL 유효성 (404, 깨진 이미지 감지)
- 등급 이미지와 메타 등급 코드 일치 여부
- 신규 배포 콘텐츠 등급 이미지 누락 여부

### 오류 대응 프로세스

```
영등위 이미지 오류 감지
    │
    ├── URL 오류 → CDN 이미지 재업로드
    ├── 등급 불일치 → 메타데이터 수정 요청
    └── 이미지 누락 → 영등위 이미지 DB에서 재매핑
    │
    ▼
수정 후 재배포 → 점검 재실행
```
