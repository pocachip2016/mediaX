# 10.3 배포/송출 관리 — 연동 인터페이스

## 수신 (Consumer)

| 제공자 | 데이터 | 이벤트 |
|--------|--------|--------|
| 3_ingest (CDN 배포) | CDN URL, 매니페스트 경로 | 인코딩 완료 후 배포 신호 |
| 1_programming/1.2_catalog | 가격 변경, 홀드백 변경 | 메타 업데이트 이벤트 |
| 1_programming/1.3_curation | 기획전·배너 편성 일정 | 편성 확정 이벤트 |
| 2_design | DA 소재 파일 | 소재 검수 완료 이벤트 |
| 5_marketing | 캠페인 일정·타겟 | 캠페인 시작 이벤트 |

## 발신 (Provider)

| 소비자 | 데이터 | 주기 |
|--------|--------|------|
| 포털 시스템 | 상품 정보, 이미지 URL, 가격 | 실시간 |
| 광고 플랫폼 | DA 소재, 게재 조건 | 캠페인 일정 |
| CDN | 퍼지 요청 | 변경 시 즉시 |
| 6_monitoring | 배포 오류 이벤트 | 실시간 |

## API 명세 (주요)

| API | 메서드 | 설명 |
|-----|--------|------|
| `/api/v1/distribution/purge` | POST | CDN 퍼지 요청 |
| `/api/v1/distribution/status/{content_id}` | GET | 콘텐츠 배포 상태 조회 |
| `/api/v1/distribution/portal-sync` | POST | 포털 데이터 즉시 동기화 |
| `/api/v1/distribution/da/submit` | POST | DA 소재 광고 플랫폼 전송 |
| `/api/v1/distribution/gtv/check` | GET | GTV 단말 배포 점검 |
