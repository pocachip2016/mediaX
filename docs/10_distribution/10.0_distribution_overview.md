# 10.0 배포 및 송출 관리 AX — 개요 (AMOC)

**미션:** 인코딩 완료된 콘텐츠를 GTV, 모바일, 웹 등 다중 단말에 안정적으로 배포하고, 영등위 이미지·포털 UI 데이터·DA 소재 전송 등 서비스 노출 전 과정을 통합 관제한다.

> **핵심 시스템:** AMOC (Asset Management & Output Control)
> **비즈니스 모듈:** M5. 배포 및 송출 관리
> **연관 기술 모듈:** 3_ingest/3.7_cdn, 6_monitoring

---

## 핵심 기능

- **플랫폼별 배포 현황** — GTV·모바일·웹 단말별 콘텐츠 배포 상태 실시간 점검
- **영등위 이미지 배포** — 등급 고지 이미지 외부 배포 상태 확인 및 오류 대응
- **콘텐츠 파일 퍼지 (Purge)** — CDN 캐시 강제 갱신, 잘못된 파일 삭제 요청
- **포털 UI 데이터 전송** — 포털 화면 구성 데이터(상품명, 가격, 이미지 URL 등) 자동 전송
- **DA 소재 전송** — 디스플레이 광고(Display Ad) 소재 플랫폼 배포

## 연동 관계

```
[3_ingest CDN 배포 완료]
         │
         ▼
[10_distribution AMOC]
    ├── GTV/단말 배포 점검
    ├── 영등위 이미지 배포 상태
    ├── Purge 요청 → CDN
    ├── 포털 UI 데이터 전송 → 포털 시스템
    └── DA 소재 전송 → 광고 플랫폼
         │
         ▼
[6_monitoring 배포 상태 모니터링]
```

## 기존 인제스트 AX와의 차이

| 인제스트 AX (3_ingest) | 배포/송출 관리 AX (10_distribution) |
|-----------------------|-----------------------------------|
| 영상 파일 수신, 인코딩, QC, DRM | 인코딩 완료 후 배포 단계 |
| CDN 업로드 및 매니페스트 생성 | 플랫폼별 배포 확인 및 퍼지 관리 |
| 파이프라인 자동화 | 운영자 수동 점검 + 자동화 혼합 |
| 기술 중심 | 비즈니스/운영 중심 |
