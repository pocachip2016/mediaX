# 미본AX - 전략, 로드맵 및 예산

## 1. 추진 전략
초기 비용 최소화를 위해 '오픈소스 우선'과 '단계적 확장' 전략을 채택합니다 [3].

## 2. 3단계 추진 로드맵
* **Phase 1 (1~3개월):** PostgreSQL DB 설계, Directus CMS 설치, TMDB/KMDb 수집, Metabase 대시보드를 구축합니다 [4]. 
* **Phase 2 (3~6개월):** Whisper 기반 자막, AI 태깅, Stable Diffusion 썸네일 서버, 마케팅 자동화를 연동합니다 [4].
* **Phase 3 (6~12개월):** pgvector 임베딩 콘텐츠 추천 및 H2O.ai 기반 예측 모델을 구축하고 클라우드 이전을 검토합니다 [4].

## 3. 예산 계획 (월 기준)
* **Phase 1:** 50~100만 원 (클라우드 및 API 종량제 비용) [5].
* **Phase 2:** 200~500만 원 (GPU 서버 및 내부 인력 포함) [5].
* **Phase 3:** 500~1,000만 원 [5].
