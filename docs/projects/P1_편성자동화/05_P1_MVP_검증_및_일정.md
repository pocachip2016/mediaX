# P1 편성 AI 자동화 - MVP 검증 및 일정

## 1. 핵심 가설
편성 작성 소요 시간을 70% 이상 단축하고, AI 추천 편성 채택률을 60% 이상 달성하는 것을 목표로 합니다 [17, 18].

## 2. 8주 개발 일정
* **W1~W3:** 환경 구성, 스키마 생성 및 외부 데이터(TMDB/KMDb 등) 파이프라인 정규화, 캘린더 CRUD 연동 [19].
* **W4~W6:** Ollama 추천 엔진, 경쟁사 패널, Airflow 자동화 연동 [19].
* **W7~W8:** 편성 담당자 파일럿 테스트 진행 및 MVP 성과 보고서 작성 [19].
