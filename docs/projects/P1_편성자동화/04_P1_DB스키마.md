# P1 편성 AI 자동화 - DB 스키마

PostgreSQL 기반의 4개 핵심 테이블을 활용합니다 [14-16].
1. **contents:** EBUCore 스키마 기반 (title_ko, content_type, genres, synopsis_ko 등) [14].
2. **schedules:** TV-Anytime 기반 (content_id, scheduled_at, slot_type, status, ai_reason 등) [15].
3. **competitor_schedules:** 플랫폼별 경쟁사 편성 크롤링 데이터 저장 [16].
4. **raw_contents:** 외부 API 수집 원본 JSON 보관 [16].
