# 1.5.5 판권계약 및 정산기준 관리 (CMT)

> **비즈니스 모듈:** M1. 파트너 및 판권 관리
> **핵심 시스템:** CMT (Content Management Tool)

## 개요

CP사와 체결하는 판권 계약의 세부 조건(정산 기준, 결제 수수료, 해외 환율, 윈도우 라이선스)을 관리한다. 계약 조건은 4.5_settlement의 정산 자동화에 직접 연동된다.

---

## 1. 판권계약 관리

### 계약 유형

| 유형 | 설명 |
|------|------|
| 독점 계약 | 자사만 서비스 가능, 타 플랫폼 배포 금지 |
| 비독점 계약 | 타 플랫폼 동시 서비스 허용 |
| 윈도우 계약 | 창구(극장→IPTV→OTT)별 서비스 기간 지정 |
| 수익 배분형 | 매출의 일정 비율 지급 |
| 고정 라이선스형 | 계약 기간 동안 고정 금액 지급 |

### 계약 항목

- 계약 기간 (시작일 ~ 종료일)
- 서비스 가능 창구 (IPTV / 모바일 / 웹 / OTT)
- 서비스 가능 지역 (국내 / 해외 국가별)
- 서비스 가능 콘텐츠 목록
- 수익 배분 비율 또는 고정 라이선스 금액
- 결제 주기 (월별 / 분기별 / 연별)
- 갱신 조건 및 우선 협상권

---

## 2. 정산기준 / 결제수수료

### 정산 기준 유형

| 정산 유형 | 계산 방식 | 적용 대상 |
|----------|----------|---------|
| RS (Revenue Share) | 매출 × 배분비율 | 대형 CP, 수익 기반 |
| MG (Minimum Guarantee) | max(보장금액, 실적 정산) | 신규 CP, 안정 수급 |
| Fixed | 월 고정금액 | 소형 CP, 채널 계약 |
| CPV (Cost Per View) | 시청 건수 × 단가 | PPV 콘텐츠 |

### 결제 수수료 구조

| 구분 | 수수료율 | 적용 조건 |
|------|---------|---------|
| 신용카드 결제 | 2.0~3.5% | 카드사별 상이 |
| 간편결제 (카카오페이 등) | 1.5~2.5% | 제휴사별 상이 |
| 통신비 합산 청구 | 0.5~1.0% | KT 통신 결합 |
| 해외 결제 (외화) | +1.0% | 해외 카드 사용 시 |

정산 기준은 CP별로 개별 설정되며, 콘텐츠 단위로도 다른 기준 적용이 가능하다.

## 3. 해외환율 관리

### 환율 기준 설정

- **기준 통화:** 원화(KRW) 기준
- **환율 기준일:** 계약서 기재 기준일 또는 정산월 평균환율
- **환율 소스:** 한국은행 기준환율 API 연동

### 해외 CP 정산 환율 처리

```
정산 기간 종료
    │
    ▼
한국은행 기준환율 API 조회 (정산월 마지막 영업일)
    │
    ▼
외화 매출액 × 기준환율 = 원화 환산 금액
    │
    ▼
계약서 환율 조건 확인 (고정환율 / 시장환율 선택)
    │
    ▼
정산서 원화 금액 확정
```

### 지원 통화

USD, EUR, JPY, CNY (계약서 기재 통화 기준)

## 4. 윈도우 라이선스 관리

### 서비스 창구(Window) 개념

콘텐츠가 출시된 후 각 서비스 채널에서 제공 가능한 시점을 순차적으로 정의하는 구조다.

| 창구 번호 | 서비스 채널 | 일반적 기간 |
|----------|-----------|-----------|
| Window 1 | 극장 | 개봉 후 90일 |
| Window 2 | PPV (건당구매) IPTV | W1 종료 후 30~90일 |
| Window 3 | 월정액/무료 IPTV | W2 종료 후 |
| Window 4 | 외부 OTT | 별도 계약 |
| Window 5 | 방송 (지상파/케이블) | 최후 |

### 윈도우 설정 기능

- CP별·콘텐츠별 창구 기간 개별 설정
- 창구 전환일 D-30/D-7/D-1 자동 알림
- 창구 전환 시 자동 가격 변경 연동 (`1.2.3_holdback` 참조)
- 창구 위반 감지 (타 플랫폼 무단 선행 서비스 감지)

## 5. 입고예정 / 제휴콘텐츠 관리

### 입고예정 콘텐츠 파이프라인

- CP사로부터 입고 예정 콘텐츠 목록 사전 수집
- 예정일, 예상 장르, 라이선스 조건 사전 등록
- 입고 완료 시 자동으로 인제스트 파이프라인 연동

### 제휴콘텐츠 관리

- 외부 OTT, 방송사, 스튜디오와의 제휴 콘텐츠 목록 관리
- 제휴 조건 (공동 제작, 독점 선공개 등) 별도 관리
- 제휴 만료 알림 및 갱신 협상 일정 관리
