# 1.1.6 VAMS — 영상 자산 가공 관리 시스템 개요

> **VAMS:** Video Asset Management System
> **비즈니스 모듈:** M2. AI 콘텐츠 및 가공 관리

## 개요

VAMS는 영상 콘텐츠의 이미지·장면·메타데이터를 AI로 자동 가공하고 외부 플랫폼에 배포하는 전용 시스템이다. 단순 메타데이터 관리(`1.1_metadata`)를 넘어, 영상에서 직접 시각 자산을 추출·가공·관리한다.

---

## 핵심 기능 영역

### 1. 영상이미지 및 장면 통합관리

| 기능 | 설명 |
|------|------|
| 키프레임 추출 | 영상에서 대표 장면 자동 추출 (시간대별, 장면 변화 기반) |
| 장면 분류 | 장면 유형 자동 분류 (액션, 대화, 풍경, 크레딧 등) |
| 이미지 통합 뷰 | 콘텐츠별 추출 이미지 전체 브라우징 및 검색 |
| 썸네일 선택 | 자동 추천 + 수동 선택 혼합 워크플로우 |
| 장면 구간 태깅 | 특정 장면 구간에 태그 부착 (Scene Search 연동) |

### 2. 부가메타 관리 (인물/그룹)

| 기능 | 설명 |
|------|------|
| 인물 DB 연동 | 출연진·감독 정보를 인물 마스터DB와 매핑 |
| 얼굴 인식 연동 | AI 얼굴 인식 결과를 인물 메타와 자동 연결 |
| 그룹 메타 | 아이돌·극단 등 그룹 단위 메타 관리 |
| 인물별 장면 목록 | 특정 인물이 등장하는 장면 자동 색인 |
| 인물 검색 API | 인물명으로 관련 콘텐츠·장면 검색 |

### 3. AI 자동화 태깅 / 외부메타 배포

| 기능 | 설명 |
|------|------|
| AI 장르 태깅 | 영상 시각 분석 기반 장르 자동 분류 |
| 감성/분위기 태깅 | 색감·음악·대사 종합 분석 (#긴장감 #따뜻한 등) |
| 외부 플랫폼 배포 | 가공된 메타·이미지를 외부 OTT/포털에 자동 배포 |
| 배포 이력 관리 | 배포 대상·시각·결과 이력 추적 |

---

## VAMS 처리 파이프라인

```
[3_ingest] 영상 원본 수신 완료
    │
    ▼
VAMS 자동 시작 (트리거 수신)
    │
    ├── 키프레임·스틸컷 추출 (썸네일 서버 → 10.7_vams_server)
    ├── 장면 분류 (AI Vision)
    ├── 얼굴 인식 → 인물 DB 매핑
    ├── 감성/분위기 분석
    └── AI 태깅 생성
    │
    ▼
가공 결과 저장 (이미지 CDN + 메타 DB)
    │
    ▼
[2_design] 썸네일 소재로 활용
[1.1_metadata] AI 태깅 결과 반영
[10_distribution] 외부 배포
```

---

## 연동 시스템

| 시스템 | 방향 | 데이터 |
|--------|------|--------|
| 3_ingest | 수신 | 영상 원본, 스틸컷 소스 |
| 1.1_metadata | 발신 | AI 태깅 결과, 인물 메타 |
| 2_design | 발신 | 키프레임 이미지 소스 |
| 10_distribution | 발신 | 가공된 메타·이미지 배포 요청 |
| 7_common_infra/MDM | 수신 | 인물 마스터 DB |
