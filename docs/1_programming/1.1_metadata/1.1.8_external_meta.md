# 1.1.8 외부 메타 수집 — 영진위 / OTT / TMDB

> **비즈니스 모듈:** M2. AI 콘텐츠 및 가공 관리 — M2-1

## 개요

CP사가 제공하는 메타데이터의 품질 보완 및 풍부화(Enrichment)를 위해 공공 데이터베이스와 외부 플랫폼에서 메타를 자동 수집·연동한다.

---

## 1. 영진위 (영화진흥위원회) 메타 수집

### 수집 데이터

| 항목 | 설명 |
|------|------|
| 영화 기본 정보 | 제목, 개봉일, 제작사, 배급사 |
| 출연진/감독 | 공식 크레딧 정보 |
| 시놉시스 | 공식 줄거리 |
| 영등위 등급 | 공식 심의 등급 및 등급 이유 |
| 관람객 수 | 박스오피스 집계 (인기도 지표) |
| 수상 내역 | 국내 영화제 수상 이력 |

### 연동 방식

- **KOBIS Open API** 사용 (영화진흥위원회 공식 API)
- 일일 업데이트 배치 (신규 등록 영화, 등급 변경)
- 콘텐츠 등록 시 자동 매핑 (제목 + 개봉연도 기반 매칭)
- 매핑 신뢰도 점수 산정 (80점 이상 자동 반영, 미만 시 담당자 확인)

---

## 2. OTT 시청 데이터 참조

### 목적

외부 OTT 플랫폼의 인기도 데이터를 참조하여 콘텐츠 수급 우선순위 및 큐레이션 의사결정에 활용한다.

### 수집 대상

| 플랫폼 | 수집 방법 | 데이터 |
|--------|----------|--------|
| 넷플릭스 | 공개 TOP10 리스트 파싱 | 주간 인기 순위 |
| 왓챠 | 공개 API / 크롤링 | 평점, 시청 인기도 |
| 웨이브 | 공개 데이터 | 주간 TOP20 |
| 티빙 | 공개 데이터 | 인기 콘텐츠 |

### 활용

- 1.5_cp_supply 수급 AI 스코어링 입력 데이터
- 1.3_curation 트렌드 감지 외부 신호

## 3. TMDB (The Movie Database) 메타 수집

### 수집 데이터

| 항목 | 설명 |
|------|------|
| 기본 메타 | 제목 (다국어), 원제, 개봉일, 런타임 |
| 장르 코드 | TMDB 표준 장르 분류 |
| 출연진 | 배역명, 배우 사진 |
| 포스터/백드롭 | 공식 홍보 이미지 (저작권 확인 필요) |
| 평점 | TMDB 사용자 평점 (인기도 지표) |
| 키워드 | 사용자 제공 키워드 태그 |
| 시리즈 정보 | 시즌, 에피소드 구조 |

### 연동 방식

- **TMDB API v3** 사용 (공식 API 키 필요)
- 콘텐츠 등록 시 자동 검색 및 매핑
- 주간 업데이트 배치 (신규 콘텐츠, 에피소드 추가)

## 4. 외부 메타 통합 처리 파이프라인

```
CP사 메타 수신 (불완전)
    │
    ▼
자동 외부 메타 조회 병렬 실행
    ├── 영진위 KOBIS API 조회
    ├── TMDB API 조회
    └── OTT 인기도 데이터 조회
    │
    ▼
메타 병합 (Merge) — 우선순위 규칙 적용
    ├── CP 제공 데이터 > 영진위 > TMDB 순
    └── 충돌 시 낮은 신뢰도 항목 → 담당자 확인 큐
    │
    ▼
품질 스코어 재산정 (1.1.2_ai_engines)
    │
    ▼
메타 DB 업데이트
```

### 우선순위 규칙

| 데이터 항목 | 1순위 | 2순위 | 3순위 |
|------------|------|------|------|
| 한국어 제목 | CP 제공 | 영진위 | TMDB (번역) |
| 원제 | TMDB | 영진위 | CP 제공 |
| 등급 | 영진위 (공식) | CP 제공 | — |
| 출연진 | 영진위 | CP 제공 | TMDB |
| 포스터 | CP 제공 | TMDB | — |
| 장르 | AI 분류 | TMDB | CP 제공 |

## 5. AI 콘텐츠 생성 — 고급 기능

> M2-4 (AI 콘텐츠 생성 및 영상 분석) 항목으로, VAMS와 연계하여 동작.

### Scene Search

- 자연어로 장면 내용을 입력하면 해당 장면 구간 검색
- 예: "비가 내리는 장면", "주인공이 울고 있는 장면"
- 기술: 영상 임베딩 벡터 + Elasticsearch 벡터 검색

### AI 자동 등급 고지

- 폭력성·선정성·언어 분석 기반 자동 시청 등급 판정
- 영등위 등급과 비교 검증 (불일치 시 경고)
- 등급 고지 이미지 자동 매핑 → 10_distribution 배포

### AI 골라보기

- 사용자 시청 패턴 기반 해당 장르·취향 핵심 장면 자동 선별
- 선별 장면으로 2~5분 미리보기 클립 자동 생성

### AI 하이라이트

- 주요 장면 자동 편집 (액션, 클라이맥스, 명대사 구간)
- 10~30초 분량 SNS 공유용 클립 생성
