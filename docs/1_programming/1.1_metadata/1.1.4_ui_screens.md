# 1.1.4 UI 화면 구성

## 화면 목록 (5개)

| # | 화면명 | 주요 기능 |
|---|--------|----------|
| 1 | 메타 대시보드 | 처리 현황 통계, 품질 분포, CP별 비교 |
| 2 | 검수 큐 | AI 제안값 vs CP 원본 비교, 원클릭 승인/수정 |
| 3 | 실시간 메타 생성 | 제목 입력 → AI 즉시 자동완성 → 확정 |
| 4 | 메타 품질 리포트 | CP사별 품질 트렌드, 자동 보정률 추이 |
| 5 | 분류 체계 관리 | 장르 트리, 태그 사전, 인물 DB 관리 |

---

## 화면 1. 메타 대시보드

- 일/주/월 처리 현황 (총 수신 / 자동 등록 / 검수 대기 / 반려)
- 품질 점수 분포 차트 (0~100점 히스토그램)
- CP별 메타 품질 비교 (레이더 차트)
- 최근 처리 콘텐츠 목록 (실시간 피드)

---

## 화면 2. 검수 큐

- 70~89점 콘텐츠 목록 (우선순위 정렬)
- 좌: CP 원본 메타 / 우: AI 제안 메타 — 항목별 diff 하이라이트
- 액션: 승인 / 수정 후 승인 / 반려 / AI 재처리 요청
- 일괄 승인 (동일 CP사 동일 장르 배치 처리)

---

## 화면 3. 실시간 메타 생성 (신규)

담당자가 제목만 입력하면 AI가 즉시 나머지 메타데이터를 채워주는 인터랙티브 화면.

### 동작 흐름

```
제목 입력 (타이핑 중 300ms debounce)
    │
    ▼
Claude API 스트리밍 응답
    ├── 시놉시스 (짧은 요약 → 상세 순서로 스트리밍)
    ├── 장르 / 세부 장르 추천 (신뢰도 % 표시)
    ├── 감성 태그 추천 (클릭으로 추가/제거)
    ├── TMDB / KOBIS 매칭 결과 (썸네일 + 기본 정보 카드)
    └── 출연진 자동 제안
    │
    ▼
담당자 검토 및 수정
    │
    ▼
"확정" 버튼 → DB 저장 + 품질 스코어 산정
```

### UI 구성 요소

| 영역 | 내용 |
|------|------|
| 상단 입력바 | 제목 + 제작연도 + CP사 선택 |
| 중앙 AI 자동완성 패널 | 스트리밍 텍스트 (타이핑 효과) |
| 우측 외부 매칭 카드 | TMDB 포스터 + 기본 정보 |
| 하단 태그 선택기 | AI 추천 태그 칩 (클릭 토글) |
| 하이브리드 모드 | 특정 필드만 "AI 추천 채우기" 버튼 (기존 CMS 방식 유지 가능) |

---

## 화면 4. 메타 품질 리포트

- CP사별 메타 품질 트렌드 (최근 90일 라인 차트)
- 자동 보정률 / 수동 검수 비율 추이
- 필드별 누락률 히트맵 (어떤 CP가 어떤 필드를 자주 빠뜨리는지)
- 월간 리포트 PDF 내보내기

---

## 화면 5. 분류 체계 관리

- 장르 트리 — 드래그앤드롭으로 계층 편집
- 태그 사전 — 동의어 그룹, 사용 빈도, AI 추천 신규 태그
- 인물 DB 관리 — 중복 인물 병합, 신규 인물 승인 큐
- CP별 장르 코드 매핑 테이블 편집
