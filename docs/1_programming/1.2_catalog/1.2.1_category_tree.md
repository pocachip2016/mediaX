# 1.2.1 서비스 카테고리 트리 관리

**구조:** 유동적 N-depth 트리 (제한 없는 계층)

**핵심 기능:**
- 드래그앤드롭 카테고리 생성/이동/병합/삭제
- 카테고리별 노출 순서 관리, 복수 카테고리 매핑
- 모듈 1 AI 분류 결과 → 카테고리 자동 매핑 제안
- 카테고리별 콘텐츠 수, 시청 성과 실시간 집계

**콘텐츠 유형:**
- **단편(Single):** 영화, 단막극, 스페셜 → 단일 상품
- **시리즈(Series):** 드라마, 예능, 애니 → 시즌 > 에피소드 계층
- 전체 구매 / 시즌 구매 / 개별 에피소드 구매 지원
