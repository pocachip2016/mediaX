# 0.1 기술 스펙 (MVP 확정)

## 프론트엔드

- **프레임워크:** Next.js 16 + TypeScript
- **모노레포:** Turbo monorepo (npm workspaces)
- **UI 패키지:** `@workspace/ui` — shadcn/ui 기반 공유 컴포넌트 라이브러리
- **스타일:** Tailwind CSS v4 (OKLch 색상 공간, CSS custom properties)
- **컴포넌트:** Radix UI primitives (`radix-ui` 통합 패키지)
- **아이콘:** Lucide React
- **테마:** next-themes (라이트/다크 모드, `attribute="class"`)
- **앱 위치:** `mediaX-CMS/apps/web/`

### 주요 패턴

| 항목 | 내용 |
|------|------|
| 라우팅 | App Router + `(main)` route group |
| 레이아웃 | `SidebarProvider` + `SidebarInset` (오버랩 없는 고정 사이드바) |
| 네비게이션 설정 | `config/docs.ts` — `NavSection[]` 트리로 중앙 관리 |
| 사이트 메타 | `config/site-config.ts` |
| 네비게이션 로직 | `lib/nav.ts` — `getActiveNav`, `getBreadcrumbs`, `getPagerLinks`, `getFlatNav` |
| 경로 별칭 | `@/*` → `apps/web/*` / `@workspace/ui/*` → `packages/ui/src/*` |

## 백엔드

- **프레임워크:** Python FastAPI
- **ORM:** SQLAlchemy + Alembic
- **작업 큐:** Celery + Redis

## AI 엔진 (외부 API — GPU 없음)

- **텍스트 분석:** Claude API 또는 OpenAI GPT-4o
- **이미지 분석:** Claude Vision 또는 GPT-4o Vision
- **얼굴 인식:** AWS Rekognition 또는 Google Cloud Vision
- **OCR:** Naver Clova OCR 또는 Google Vision OCR

## 데이터베이스

- **메인 DB:** PostgreSQL
- **검색:** Elasticsearch
- **캐시:** Redis

## 작업 큐

- Celery + Redis

## 인프라 (로컬 MVP)

- Docker Compose / Nginx / Grafana + Prometheus

## 개발 환경

- 팀 1~2명, Python + JS/TS, 로컬 서버
- Node.js 22 (nvm, `nvm use 22`)

## 프로젝트 구조

│   ├── api/
│   ├── workers/
