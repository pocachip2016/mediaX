# §2. 소스 우선순위 (한국 VOD 특성 반영)

> 소속: Phase C ADR — `_index.md` | 인접: §5 [beat-schedule.md](beat-schedule.md), §6 [ops-cost.md](ops-cost.md)

지니TV 의 시청자 분포(한국 영화·드라마 70%, 글로벌 30%)와
편성 운영의 *발굴 가치* 를 함께 고려해 우선순위를 정한다.

## 2.1 우선순위 표

| 우선 | 소스 | 발굴 영역 | 신뢰도 | 빈도 |
|---|---|---|---|---|
| 1 | **TMDB** | 글로벌 신작·트렌딩 (Trending day/week, Upcoming, Discover) | 0.85 | daily |
| 2 | **KOBIS** | 한국 개봉예정작·박스오피스·신규 등록 | 0.95 | daily |
| 3 | **KMDB** | 한국 영화·시리즈 풀백필·과거작 | 0.92 | daily |
| 4 | **OMDb** | IMDb 글로벌 보완 (TMDB 누락분, 영문 메타) | 0.80 | on-demand |

신뢰도 컬럼은 SEED 적재 시 `base_confidence` 초기값. Phase B `compute_match_score`
의 가중과 일치 (`scoring.py:23-24`) — 동일 정책의 **다른 사용처** 일 뿐, 별도 설정 아님.

## 2.2 소스별 발굴 영역 매트릭스

| 영역 | TMDB | KOBIS | KMDB | OMDb |
|---|---|---|---|---|
| 글로벌 신작 영화 | ✅ 1차 | – | – | 보완 |
| 한국 개봉예정 영화 | – | ✅ 1차 | 보완 | – |
| 한국 박스오피스 (현재 상영) | – | ✅ 1차 | – | – |
| 한국 영화 풀백필 (1990~) | – | – | ✅ 1차 | – |
| 글로벌 TV 시리즈 (시즌/에피소드) | ✅ 1차 | – | – | 보완 |
| 한국 TV 시리즈 | ✅ 1차 (TMDB-KO) | – | 보완 | – |
| IMDb ID 보완 (해외 평점·리뷰 연결) | – | – | – | ✅ 1차 |

"1차" = 해당 영역의 발굴 책임 소스. "보완" = 검수자가 on-demand 로 추가 호출.

## 2.3 우선순위가 정해지는 이유

- **TMDB 1순위 (글로벌 신작)**: API 안정성·갱신 빈도·시리즈 계층 표현이 가장 우수.
  Phase B 인프라(tmdb_client.py, content_credits)가 이미 TMDB 스키마 기준.
- **KOBIS 2순위 (한국 개봉예정)**: 정부 공식 데이터로 신뢰도 최상이지만
  TV 시리즈·VOD 미수록 → 영화로 한정.
- **KMDB 3순위 (백필)**: 풀백필 가치는 높으나 신작 갱신은 KOBIS 가 빠름.
- **OMDb 4순위 (on-demand)**: free tier 1000/day 제한 → 일괄 발굴 부적합,
  검수 화면에서 IMDb ID 채울 때만 호출.
