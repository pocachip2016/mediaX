## 화면 1 — 편성 보드 `/programming/scheduling`

3컬럼. 중앙이 LINK 편집기, 우측이 노드 속성 + 다중부모 역참조.

```
┌─ 편성 보드 ───────────────────────────[편성안: 2026 여름 ▾] [저장] [발행]─┐
│ 좌: 소스 팔레트         │ 중앙: 편성 캔버스(선택노드의 LINK)│ 우: 노드 속성  │
│─────────────────────────│───────────────────────────────────│───────────────│
│ [작품][노드][계층]탭     │ 📍홈 > 여름특집 > 마블 특집        │ ◆ 마블 특집   │
│ 🔍 검색________________  │   (등장 위치 3곳 ▾)               │ kind: manual  │
│ 필터:장르▾ 연도▾ CP▾    │───────────────────────────────────│ headline:     │
│ ▸ 작품 풀               │ 자식 LINK (드래그 정렬)           │ [여름엔 마블] │
│   🎬 어벤져스           │ ⠿1 [콘텐츠]🎬어벤져스 📌 ▢기간    │ sub:[_______] │
│   📺 로키 └S1 └S2       │ ⠿2 [콘텐츠]🎬아이언맨   ▢기간     │ theme:🏷여름  │
│ ▸ 재사용 노드           │ ⠿3 [노드]  📁시즌2모음 ▸전개      │ 노출:06/01~08/31│
│   📁 액션TOP10(rule)    │ ⠿4 [콘텐츠]🎬토르      ▢기간      │ ───────────── │
│   📁 가을기획(manual)   │ [+작품 끌어놓기][+노드 참조][CSV]  │ ▼ 등장 위치   │
│─────────────────────────│ ⚠ 사이클 가드: 자기참조 불가      │ •홈1행 •영화관│
│ [편성안 목록 ◀]         │                                   │ •여름특집     │
└──────────────────────────────────────────────────────────[미리보기 ▶]──┘
```

- 중앙: 콘텐츠+하위노드 혼합 자식을 드래그 정렬, 행별 📌pin/▢기간 override.
- 우측 하단 "등장 위치": 다중부모 **역참조** — 이 노드가 어디 걸렸는지.
- kind=rule 선택 시 우측이 쿼리빌더+[🤖AI추천], rank면 지표/N 입력으로 전환.

## 화면 2 — AI 추천 패널 (우측 패널 확장)

```
┌─ ◆ 마블 특집 [kind: rule] ────────────────────────────┐
│ 의도(자연어):[여름에 어울리는 시원한 마블___][🤖 생성] │
│ AI 추천 (source=ai·미확정)            정렬: 적합도▾    │
│ ☑🎬어벤져스   0.94 근거:액션·여름블록버스터·인기      │
│ ☑🎬토르 천둥  0.88 근거:히어로·유머·시즌적합          │
│ ☐🎬이터널스   0.52 근거:평점 낮음(제외 권장)          │
│ [선택 확정][전체 확정][재생성]  신뢰도<0.6 자동제외☑   │
└──────────────────────────────────────────────────────────┘
```

근거 한 줄 필수(설명가능성) → 운영자 일괄 승인 신뢰.

## 화면 4 — 편성 그래프/영향도 (DAG 검증)

```

      │             └──▶시즌2모음 ─┐
```
