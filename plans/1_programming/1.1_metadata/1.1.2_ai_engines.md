## 세션 분할

### 세션 1 — Claude API 프로바이더 추가
범위:
- `backend/api/programming/metadata/llm/claude.py` 신규 작성
  - `anthropic` SDK 사용 (`pip install anthropic`)
  - 인터페이스: 기존 `gemini.py` 패턴 동일하게
- `ai_engine.py` 폴백 체인 재조정: Claude → Gemini → Groq → Ollama
- `.env.example`에 `ANTHROPIC_API_KEY=` 추가
- 완료 기준: `python3 -c "from api.programming.metadata.llm.claude import ClaudeProvider; print('OK')"`

### 세션 2 — 스트리밍 API + 프론트 UX
범위:
- `router.py`에 `POST /metadata/text/suggest/stream` 스트리밍 엔드포인트 (StreamingResponse)
- `lib/api.ts`에 `textMetaApi.suggestStream` (EventSource 또는 fetch stream)
- `text/page.tsx` 스트리밍 UX (debounce 300ms + 타이핑 애니메이션)
- 완료 기준: 브라우저에서 텍스트가 실시간으로 채워지는 것 확인

### 세션 3 — CLIP 감성 태깅 (Phase 3, 조건부)
범위: VAMS 썸네일 구현 후 착수
- `open-clip-torch` 설치
- 포스터 벡터화 → 감성 태그 후보군 코사인 유사도 비교
- 텍스트 분석 결과와 앙상블
