아래 순서로 진행해줘.

## Step 1 — 플랜 파일 선택

**인자가 없으면** (즉 $ARGUMENTS가 비어있으면):
```bash
ls plans/
```
를 실행해서 목록을 보여주고, 어떤 파일로 시작할지 물어봐. 그 다음은 진행하지 마.

**인자가 있으면** `plans/$ARGUMENTS.md`를 읽어. (이 파일만, docs/ 탐색 금지)

## Step 2 — 다음 세션 자동 감지

플랜 파일의 "세션 분할" 섹션에서 체크박스(`- [ ]`)가 남아있는 **첫 번째 세션**을 찾아.
- 모두 완료(`- [x]`)면 "모든 세션 완료" 보고 후 종료.
- 찾은 세션 번호와 범위를 한 줄로 선언: **"세션 N — [범위]를 시작합니다. 읽을 파일: [목록]"**

## Step 3 — 구현

1. 패턴 참조 파일은 plans에 명시된 파일만, Grep으로 줄번호 확인 후 필요한 함수만 읽기
2. 선언한 세션 범위 외 파일 수정 금지
3. 구현 완료 후 pytest 또는 `python3 -c "from X import Y; print('OK')"` 실행
4. 완료 후 플랜 체크박스 업데이트
